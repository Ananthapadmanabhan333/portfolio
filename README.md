## ✨ Features

- **🎨 Deep Cherry Velvet Design** — Rich dark theme with crimson-burgundy glassmorphism and ambient glow orbs
- **🖼️ Authentic Photo Integration** — Personal portrait with parallax hover effect and editorial framing
- **⌨️ Animated IDE Terminal** — Real-time code typing simulation that prints academic and skills data
- **🃏 3D Magnetic Card Tilt** — GPU-accelerated perspective transforms on all project cards
- **🔍 Filterable Project Index** — 27 repositories organized across 5 categories with live tab filtering
- **📱 Fully Responsive** — Adaptive layout from desktop (1400px) to mobile (320px)
- **♾️ Scroll Reveal System** — Intersection Observer-based entry animations for every section
- **🔗 GitHub Redirects** — Every project card opens the exact repository in a new tab

## 🧰 Tech Stack

```
Structure  →  Semantic HTML5
Styling    →  Vanilla CSS3 (Custom Properties, Grid, Flexbox, Animations)
Logic      →  Vanilla JavaScript (ES6+, IntersectionObserver, EventListeners)
Fonts      →  Google Fonts (Cormorant Garamond + Inter)
Icons      →  Inline SVG
Server     →  Python HTTP server (local dev)
```

## 🚀 Run Locally

```bash
# Option 1: Python
python -m http.server 8080

# Option 2: Node
npx serve .

# Option 3: VS Code — use Live Server extension
```
